Because GPUs are scarce and prices are soaring, I’ve made the most inefficient deal finder I could think of. Instead of a simple bot updating common stores and looking at the price, I’ve made a bot that will query a bunch of different AIs, asking them to do a current web search and do some calculations and research to figure out if the current price is a bargain.

I have also picked up on the popularity of “idle” games. So, I made a Cookie Clicker knock-off, called “AI kicker”. The goal of the game is to gather tokens, by interacting with AI/LLM websites and try to prompt the point scoring answer. With these tokens, you can purchase “jr. Prompt engineer” etc. to increase your token rate. The best part of this game is that it’s entirely run by a local AI, making it truly idle.

And finally, what’s gaming without music? I’m running an agent that will create a song about everything as it’s happening in real-time. Using logs from both the idle game and deal finding to create automated prompts with the help of music LLM services, of course.
